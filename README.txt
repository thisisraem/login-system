Title: Responsive Login Form with Social Media Authentication

Description: This app provides a responsive login form with options to authenticate 
using email and password or through popular social media platforms like Google and Facebook. 
The form is designed with a clean and modern user interface, making it easy for users to log in securely. 
Users can enter their email and password in the designated fields or choose to sign in with their Google or Facebook accounts. 
The form includes proper validation for required fields and utilizes the Django framework for form submission and authentication.
With this app, users can conveniently access your platform using their preferred login method, 
enhancing the user experience and improving the security of your application.
